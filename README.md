Este repositório contém o código-fonte do meu portfólio pessoal, um espaço criado para centralizar a minha evolução técnica, projetos e trajetória acadêmica. Desenvolvido com HTML, CSS e JavaScript puros, sem frameworks ou dependências de build complexas. O objetivo foi construir uma interface ágil e responsiva para destacar de forma direta meu percurso universitário, minha vivência profissional e meus projetos de desenvolvimento.

🔗 [Ver online](https://portfolio-maria-clara.vercel.app/)

---

## ✨ Seções

* **Sobre** — Apresentação com nome, cargo, CTA para currículo e destaques profissionais.
* **Tecnologias** — Grid interativo com as principais ferramentas e linguagens utilizadas.
* **Experiência** — Timeline detalhada com experiências.
* **Projetos** — Cards com descrição, tecnologias utilizadas e links fixos para repositórios.
* **Formação Académica** — Resumo da trajetória no Instituto Metrópole Digital (IMD).
* **Contato** — Links diretos para GitHub, LinkedIn e e-mail com protocolo mailto.

---

## 🛠️ Tecnologias Utilizadas

* **HTML5** — Estrutura semântica para melhor SEO e acessibilidade.
* **CSS3** — Estilização moderna utilizando Variáveis (Custom Properties), Flexbox e CSS Grid para layouts complexos.
* **JavaScript (Vanilla)** — Lógica personalizada para menu mobile, fecho de elementos ao clicar fora e interatividade de toque.
* **AOS (Animate On Scroll)** — Biblioteca para animações de entrada fluidas durante o scroll.
* **Font Awesome & Devicon** — Ícones de alta qualidade para tecnologias e interface de utilizador.
* **Google Fonts (Segoe UI/System-UI)** — Tipografia limpa e legível.
